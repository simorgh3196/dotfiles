# Dotfiles

## Installation

```sh
$ make install  # install all
```

or

```sh
$ make install/nvim  # install selection dotfiles
$ make install-...
```

## Dependencies

fish depends on
- peco
- ghq
- gh-open


## Install list

- git
- tmux
- nvim
- fish
